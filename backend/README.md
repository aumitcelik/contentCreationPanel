#  Locally Installation
```bash
 yarn
```

## Running the app
**You have to postgresql installed on your device. Edit properties on backend/.env**
```bash
 yarn start
```
Started on http://localhost:5000

### Testing
#### Unit testing
**On backend**
```bash
yarn test
```
#### e2e api testing
**On backend**
Install dependencies
```bash
yarn
```
Start the backend locally.
```bash
yarn start
```
Start  test
```bash
yarn test:e2e
```
